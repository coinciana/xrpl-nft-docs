# XRPを入手する

XRPL NFTを利用するには仮想通貨 **XRP** （エックスアールピー）が必要です。  
多くの国内仮想通貨取引所で購入することができます。

- [SBI VCトレード](https://www.sbivc.co.jp/)
- [bitbank](https://bitbank.cc/)
- [Coincheck](https://coincheck.com/ja/)
- [GMOコイン](https://coin.z.com/jp/)
- [DMM Bitcoin](https://bitcoin.dmm.com/)


<a id="01_01_01"></a>
## XRPL NFTを始めるのに必要なXRP

XRPL NFTを触るのに必要なXRPは **15XRP** 程度です。


<a id="01_01_02"></a>
## XRPL NFTで使用するXRP

- ### アドレスの有効化（アクティベート）　10XRP
xummでウォレットアドレスを作成した後、アドレスを有効化するのに10XRPのデポジットが必要です。  
アドレスを放棄すると取り戻すことができます。

- ### オファーの所有　2XRP（1オファーあたり）
オファーをひとつ出すごとに、準備金として一時的に2XRPがロックされ、使用不可能になります。  
オファーが処理またはキャンセルされると2XRPはウォレットに戻ります。  
2XRPはアカウントの残高から実際に差し引かれることはありませんが、ウォレットや利用サービスによっては利用可能な残高として差し引かれた残高が表示されることがあります。

- ### NFTの所有　2XRP（32点ごと）
NFTを約32点保有するごとに、準備金として一時的に2XRPがロックされ、使用不可能になります。  
NFTを手放すと2XRPはウォレットに戻ります。  
2XRPはアカウントの残高から実際に差し引かれることはありませんが、ウォレットや利用サービスによっては利用可能な残高として差し引かれた残高が表示されることがあります。

- ### トランザクション手数料　0.000015XRP（1TXあたり）  
送金やMintにかかるガス代のようなもの。

参考　[XRPLドキュメント　準備金](https://xrpl.org/ja/reserves.html)
